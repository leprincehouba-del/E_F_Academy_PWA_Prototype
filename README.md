# E. F Academy — PWA Prototype

نسخة تجريبية لتطبيق إدارة أكاديمية اللغة الإنجليزية.

## بيانات الدخول التجريبية
- رقم الهاتف: 01000000000
- كلمة المرور: 123456

## التشغيل
لأن تثبيت الـ PWA وService Worker يحتاجان فتح المشروع من خادم محلي:

### على الكمبيوتر
1. فك ضغط الملف.
2. افتح Terminal / Command Prompt داخل مجلد المشروع.
3. شغّل:
   python -m http.server 8080
4. افتح في المتصفح:
   http://localhost:8080

يمكن أيضًا فتح `index.html` مباشرة لمشاهدة الواجهات، لكن التثبيت والعمل دون إنترنت يحتاجان الخادم المحلي أو استضافة حقيقية.

## ما يعمل في النسخة
- تسجيل دخول تجريبي.
- لوحة إدارة متجاوبة.
- الجدول المعتمد للمجموعات.
- تسجيل الحضور والغياب والتأخير.
- حساب سعر الحصة حسب المرحلة.
- تراكم حتى 3 حصص مع منع التجاوز إلا بموافقة الإدارة.
- تسجيل المدفوعات.
- نظام Points:
  - حضور +3
  - غياب -10
  - تأخير -2
  - واجب +5
  - مشاركة 1–10
  - امتحان حتى 20 حسب الدرجة
- واجهة ولي الأمر.
- فتح واتساب برسالة جاهزة.
- إضافة طالب وحفظ البيانات محليًا في المتصفح.

## ملاحظة مهمة
الإرسال الآلي على واتساب وتسجيل الحسابات الحقيقية وقاعدة البيانات المركزية تحتاج:
- Backend
- قاعدة بيانات
- استضافة
- WhatsApp Business API
- نظام صلاحيات وأمان
